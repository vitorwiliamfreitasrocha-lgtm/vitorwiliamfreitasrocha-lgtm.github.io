Informações Gerais

O VentraSense é uma plataforma de monitoramento meteorológico desenvolvida para uso profissional em aviação, agronegócio e ambientes industriais. Esta política descreve como coletamos, usamos e protegemos seus dados.

Ao utilizar nossos serviços — site, aplicativo móvel ou dashboard web — você concorda com os termos descritos neste documento.

Dados Coletados

O aplicativo e a plataforma coletam as seguintes informações para seu funcionamento:

Credenciais de acesso: nome de usuário e senha para autenticação. As senhas nunca são armazenadas em texto puro — utilizamos hash criptográfico padrão da indústria.

Dados meteorológicos: temperatura, umidade, velocidade e direção do vento, precipitação e pressão atmosférica coletados pelas estações VentraSense cadastradas.

Token de sessão: um token JWT é armazenado localmente no dispositivo para manter a sessão ativa de forma segura.

Dados de contato: e-mail e nome informados via formulário de contato, utilizados exclusivamente para atendimento comercial.

Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticar o usuário e garantir acesso seguro às estações cadastradas;
  • Exibir dados meteorológicos em tempo real no dashboard e aplicativo;
  • Gerar histórico de medições e disparar alertas configurados pelo próprio usuário;
  • Responder solicitações comerciais e de suporte técnico.

Compartilhamento de Dados

O VentraSense não vende, aluga ou compartilha dados pessoais com terceiros para fins comerciais. Os dados meteorológicos são transmitidos exclusivamente entre a estação contratada, o servidor da VentraSense e o dispositivo autorizado do usuário.

Exceção legal
Dados podem ser compartilhados apenas em cumprimento a obrigação legal, ordem judicial ou requisição de autoridade competente — conforme previsto no Art. 7º da LGPD.

Segurança

Toda a comunicação entre o aplicativo, o dashboard e o servidor é realizada via HTTPS (TLS 1.3) com autenticação por token JWT. Os dados são armazenados em servidores seguros, com backups criptografados e acesso restrito a pessoal autorizado.

Adotamos boas práticas de segurança da informação alinhadas com a LGPD e padrões internacionais (ISO 27001).

Retenção de Dados

Os dados meteorológicos são retidos por até 30 dias no servidor para usuários do plano padrão. Planos Aero e Enterprise podem ter período de retenção estendido conforme contrato.

O token de sessão é removido automaticamente ao realizar logout ou após expiração. Dados pessoais são excluídos mediante solicitação formal pelo titular.

Direitos do Usuário

Conforme a LGPD, o titular dos dados possui os seguintes direitos:

  • Confirmação da existência de tratamento de dados;
  • Acesso aos dados pessoais coletados;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Anonimização, bloqueio ou eliminação de dados desnecessários;
  • Portabilidade dos dados a outro fornecedor;
  • Revogação do consentimento a qualquer momento.

O aplicativo não coleta dados de menores de 18 anos. Para exercer qualquer direito, entre em contato pelo e-mail abaixo.
